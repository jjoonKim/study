# 깃 허브 연결

1. '터미널' 을 열고 Git Bash 에서 아래와 같이 입력
```
// 깃 초기화
git init
// 깃 사용자 등록    
git config user.name'유저네임'
// 깃 이메일 등록 
git config user.mail'유저이메일'
```
![img.png](이미지/img.png)
체크된 박스를 복사 하여 Git Bash 에 붙여넣기 한다.
